B) Directories & File Uses

Our Design uses only assets & lib folder. So, don't touch any other folder for ui changes.

    • assets(Folder) - Put all Static images & other data (in Our Case images & json file) in this folder
    • fonts(Folder) - Put all font file in this folder
    • lib(Folder) - Main folder that contains all the dart code
      • appBehaviour(Folder) - Contain App Behaviour (disable ripplr effect) in this folder
      • constant(Folder) - Contain Constants (fixPadding, textstyle, google map api key, etc) in this folder
      • Pages(Folder) - Contain All App Pages in this folder except main.dart
        • about(Folder) - Contain about us page in this folder
        • appointment(Folder) - Contain all appointment related file in this folder
        • chat(Folder) - Contain all chat related file in this folder
        • doctor(Folder) - Contain all doctor related file in this folder
        • home(Folder) - Contain home page in this folder
        • lab(Folder) - Contain lab related file in this folder
        • login_signup(Folder) - Contain login, otp screen, register page in this folder
        • notifications(Folder) - Contain notification page in this folder
        • patient(Folder) - Contain patient directory page in this folder
        • payment(Folder) - Contain payment page in this folder
        • profile(Folder) - Contain profile and edit profile page in this folder
        • review(Folder) - Contain review page in this folder
        • search(Folder) - Contain search page in this folder
        • speciality(Folder) - Contain speciality page in this folder
      • bottom_bar.dart(Page) - Contain bottom bar of home page.
      • screens.dart(Page) - Export all screen in this file.
      • splashScreen.dart(Page) - Contain splash screen code.
    • main.dart - Entry Point of Our App.
  • pubspec.yaml(File) - Specification file (All the package and local assets (Like Images, Fonts, etc.) we use in this application is specify in this file.).
